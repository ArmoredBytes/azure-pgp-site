About

The azure-pgp tool provides opinionated commands to generate OpenPGP keypairs and persist them in Azure Key Vault, enabling secure distribution and usage across your cloud workloads.

  • Cross-platform binaries
  • Uses Azure Key Vault for secure storage
  • Automates key lifecycle tasks
